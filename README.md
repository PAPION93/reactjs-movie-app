## 1. ReactJS

1.0. 세팅

-   [node.js 설치](https://nodejs.org/ko/) (npm)
-   npx 설치(npm install npx -g)
-   Git 준비
-   VScode 사용

1.1. 생성 및 실행

-   `npx create-react-app tell-me-what-u-want`
-   In VScode `Open Folder`
-   `npm start`

1.2. ReactJS

-   Virtual DOM
-   Component
-   JSX
-   prop-types 유효성검사
    -   `npm i prop-types`
-   axios API 요청
    -   `npm i axios`
-   router
    -   `npm install react-router-dom`
-   redux (state save)
